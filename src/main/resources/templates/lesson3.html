Basic - basic Springboot 1 (3/4)
Setelah melakukan konfigurasi, kita perlu membuat sebuah class yang berisi entity dari tabel database yang telah terkoneksi seperti berikut :

Annotation @Entity menandakan bahwa class ini merupakan class Entity.
Annotation @Id menandakan bahwa kolom tersebut adalah Primary Key.
Annotation @GeneratedValue menandakan bahwa kolom ini menggunakan auto generated value sebagai isi dari kolom tersebut.
Annotation @Column menandakan kolom yang dituju pada tabel database.
Parameter “name” pada @Column adalah nama kolom pada tabel database.
Setiap kolom harus memiliki deklarasi variable pada class entity yang akan digunakan untuk pengolahan data pada aplikasi.
Setelah itu, seperti konsep OOP (Object Oriented Programming), diperlukan adanya constructor, getter dan setter untuk setiap variable.
