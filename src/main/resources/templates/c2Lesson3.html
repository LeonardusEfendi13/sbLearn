Memulai dengan Springboot (3/3)
Untuk memulai sebuah aplikasi menggunakan framework Springboot, kita hanya perlu masuk ke website https://start.spring.io/ yang dimana halaman website tersebut akan terlihat seperti ini :


Pada bagian “Project”, kita perlu menentukan build manager apa yang akan digunakan untuk aplikasi kita, terdapat beberapa pilihan diantaranya “Gradle-Groovy”, “Gradle-Kotlin”, dan “Maven”.

Pada bagian “Language”, kita dapat menentukan bahasa pemrograman untuk aplikasi kita, baik Java, Kotlin, maupun Groovy”.

Pada bagian “Spring Boot”, kita akan menentukan versi framework Springboot yang akan digunakan sesuai keinginan kita.

Pada bagian “Project Metadata” akan mengisi detail dari setiap field seperti ingin membuat project dalam Java, dan diakhir akan menentukan versi Java sesuai keinginan.

Setelah semua terisi dan terpilih, langkah selanjutnya adalah menambah dependency yang diperlukan sesuai kebutuhan aplikasi.

Langkah terakhir adalah menekan tombol “Generate” dan framework Springboot akan terunduh dalam format Zip, kemudian kita hanya perlu mengekstrak file tersebut dan membukanya menggunakan IDE masing-masing.
