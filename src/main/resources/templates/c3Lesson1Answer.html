Database dan Springboot (1/4)
Untuk melakukan koneksi ke database, diperlukan dependency untuk mendukung koneksi tersebut. Contoh dependency yang sering digunakan adalah dependency JPA (Java Persistance API) dan JDBC (Java Database Connectivity). Setelah menambahkan dependency untuk koneksi ke database, kita perlu menambahkan dependency untuk database yang dituju (MySQL, Oracle, dan lain-lain). Untuk masing-masing jenis database memiliki dependency yang berbeda-beda, contohnya adalah sebagai berikut :

1. MySQL

2. Oracle

Perlu diketahui bahwa untuk penggunaan dependency-dependency tersebut dapat berbeda-beda, maka saat ingin menambah dependency lebih baik melihat melalui dokumentasi resmi masing-masing database.
